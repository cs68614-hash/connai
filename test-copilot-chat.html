🤖 ConnAI Copilot Chat 功能测试

⚠️ 重要提示：

测试此功能前，请确保：

  • VS Code 已安装并运行 ConnAI 扩展
  • VS Code 已安装 GitHub Copilot Chat 扩展
  • 浏览器已安装 ConnAI 浏览器扩展
  • VS Code 协议服务器正在运行（通常在端口 6797）

🔧 测试步骤

  1. 准备环境：
    • 在 VS Code 中打开一个项目
    • 确保 ConnAI 扩展已激活
    • 确保 Copilot Chat 扩展已登录
  2. 安装浏览器扩展：
    • 在 Chrome 中加载 packages/browser-extension/.output/chrome-mv3 文件夹
    • 确保扩展已启用并且有侧边栏权限
  3. 测试聊天功能：
    • 在下方文本框中输入一些文字
    • 右键点击文本框，选择 "💬 Open Copilot Chat"
    • 观察浏览器侧边栏是否打开
    • 在侧边栏中输入问题并测试与 Copilot 的对话

🧪 测试检查点

成功标准：
  • ✅ 右键菜单显示 "💬 Open Copilot Chat" 选项
  • ✅ 点击菜单项后，浏览器侧边栏自动打开
  • ✅ 侧边栏显示 Copilot Chat 界面
  • ✅ 可以在侧边栏中输入问题
  • ✅ Copilot 能够响应并返回流式答案
  • ✅ 聊天记录正确显示
  • ✅ 连接状态正常显示

🔧 故障排除

如果遇到问题：
  • 菜单不显示： 检查浏览器扩展是否正确加载
  • 侧边栏不打开： 检查浏览器是否支持 sidePanel API
  • 连接失败： 检查 VS Code 扩展和协议服务器状态
  • Copilot 不响应： 检查 Copilot Chat 扩展登录状态
  • 流式响应问题： 检查浏览器控制台和 VS Code 输出面板

📋 开发调试信息

调试工具：
  • 浏览器： F12 开发者工具 → Console
  • VS Code： 查看 → 输出 → ConnAI
  • 扩展： 扩展页面 → ConnAI → 检查视图
  • 协议服务器： 检查端口 6797 是否可访问
